# kamori's tour — Design System

프로젝트의 디자인 토큰이 Storybook 으로 로드되었습니다. 상단 툴바와 우측 패널에서 아래 3가지를 전환하며 확인해 보세요.

## 🔎 확인 사항

- **🌐 Language (상단 툴바)** — 5개 언어 전환. 언어 바꾸면 `<html lang>` 이 갱신되고 `global.css` 의 `html[lang]` 셀렉터에 의해 폰트가 자동 교체됩니다.
  EN · 日本語 · 한국어 · Español · 中文
- **📐 Viewport (상단 툴바)** — mobile 375 / tablet 768 / desktop 1280 / large 1440 (layout-system §3)
- **🎨 Backgrounds (상단 툴바)** — white · off-white · silver-veil · ink + 파스텔 tint 3종
