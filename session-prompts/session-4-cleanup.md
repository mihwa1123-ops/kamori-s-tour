## ⚠️ 자주 발생하는 문제

### 문제 1: Storybook 사이드바 순서 이상함
**증상**: Tag, Badge, Button 순서가 알파벳 안 따름  
**해결**:
```
각 .stories.tsx 의 meta.title 이 정확히 'Atoms/ComponentName' 인지 확인.
Storybook 은 알파벳 순서로 자동 정렬.
```

### 문제 2: autodocs 페이지가 안 생성됨
**증상**: 컴포넌트 옆에 'Docs' 탭 없음  
**해결**:
```
meta.tags 에 ['autodocs'] 가 있는지 확인.
.storybook/main.ts 의 docs.autodocs 가 'tag' 인지 확인.
```

### 문제 3: 빌드는 되는데 콘솔에 에러
**증상**: 브라우저 콘솔에 React key 경고  
**해결**:
```
모든 .map() 에 unique key prop 있는지 확인.
배열 index 를 key 로 쓰면 안 됨, id 같은 고유 값 사용.
```

### 문제 4: 빌드 사이즈 너무 큼
**증상**: dist/ 폴더가 5MB 초과  
**해결**:
```
import 분석:
- 사용하지 않는 컴포넌트 import 제거
- 트리쉐이킹 적용 확인 (named import 사용)
- 폰트 불필요한 weight 로드 안 하는지

목표: 메인 번들 < 200KB (gzip)
```
